# Blogri

Post handler application with google authentcation, light and dark mode.

**Stack:**

* NextJS
* React
* Tailwind
* Firestore

**Features:**

* [X] Login with google
* [X] Get posts and comments from Firestore database
* [X] Full CRUD for posts
  * [X] List
  * [X] Create
  * [X] Edit
  * [X] Delete
* [ ] Full CRUD for comments
  * [X] List
  * [X] Create
  * [ ] Edit
  * [ ] Delete
